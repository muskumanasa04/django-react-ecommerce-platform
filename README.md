# Django-React Ecommerce Platform

Full-stack ecommerce platform built with **Django REST Framework** (backend) and **React.js** (frontend), implementing **JWT authentication**, **product management**, **cart functionality**, and **order tracking**.

## Features

###  User Functionality
- User sign-up and login with JWT authentication
- Browse products with search & filtering
- Add products to cart
- Place orders
- View order history & details

###  Cart & Checkout
- Add/update/remove cart items
- Calculate totals dynamically
- Checkout process

###  Backend (Django)
- RESTful API with Django REST Framework
- JWT token auth (`access` & `refresh`)
- Product CRUD operations
- Order management

###  Frontend (React)
- React components for store pages
- State management (e.g. Redux, Context API)
- API integration with backend
- Responsive UI


## 🛠 Tech Stack

| Layer      | Technology |
|------------|-------------|
| Backend    | Django, Django REST Framework |
| Auth       | JWT Authentication |
| Frontend   | React.js, Axios / Fetch |
| Database   | SQLite / PostgreSQL |
| Styles     | CSS / UI Library (optional) |
